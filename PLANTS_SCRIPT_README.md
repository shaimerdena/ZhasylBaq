# ЖасылБақ - Скрипт управления растениями

## Описание
Скрипт `plants.js` обеспечивает полное управление растениями в приложении ЖасылБақ, включая:
- ✅ Добавление новых растений
- ✅ Сохранение данных в localStorage
- ✅ Автоматическое отображение растений на странице
- ✅ Обновление статистики
- ✅ Удаление растений

## Функции

### Основные функции:

#### `addPlant(plantData)`
Добавляет новое растение в хранилище.

**Параметры:**
```javascript
{
  name: "Алма ағашы",        // Название растения
  type: "Жеміс ағашы",        // Тип растения
  sector: "А-01",             // Сектор размещения
  plantedDate: "2025-04-15",  // Дата высадки
  waterFrequency: "Аптасына бір рет", // Частота полива
  notes: "Жарықтан қорғау керек"      // Дополнительные заметки
}
```

#### `deletePlant(plantId)`
Удаляет растение по ID.

#### `updatePlant(plantId, updates)`
Обновляет данные растения.

#### `renderPlants()`
Отображает все растения на странице.

#### `updateStats()`
Обновляет статистику (количество здоровых, требующих внимания и критичных растений).

## Использование

### 1. Добавление растения через форму
Кнопка "＋ Жаңа өсімдік қосу" открывает модальное окно с формой. Заполните поля:
- **Название** - выберите или введите название растения
- **Тип** - выберите категорию
- **Сектор** - укажите место расположения (А-01, Б-05 и т.д.)
- **Дата высадки** - выберите дату
- **Частота полива** - выберите периодичность
- **Заметки** - добавьте дополнительную информацию

Нажмите "✓ Қосу" для добавления.

### 2. Программное добавление растения
```javascript
// Пример в консоли браузера:
addPlant({
  name: 'Роза',
  type: 'Гүл',
  sector: 'Б-03',
  plantedDate: '2025-04-10',
  waterFrequency: 'Күнде бір рет',
  notes: 'Красивый цветок'
});
```

### 3. Просмотр всех растений
```javascript
const plants = getPlantsFromStorage();
console.log(plants);
```

### 4. Обновление растения
```javascript
updatePlant(plantId, {
  health: 85,
  moisture: 65,
  temperature: 22
});
```

### 5. Удаление растения
```javascript
deletePlant(plantId);
```

## Структура данных растения

Каждое растение хранится со следующими свойствами:
```javascript
{
  id: 1713206400000,                    // Уникальный ID (timestamp)
  name: "Алма ағашы",                  // Название
  type: "Жеміс ағашы",                 // Тип
  sector: "А-01",                       // Сектор
  plantedDate: "2025-04-15",           // Дата высадки
  waterFrequency: "Аптасына бір рет",  // Частота полива
  notes: "Жарықтан қорғау керек",      // Заметки
  emoji: "🍎",                          // Эмодзи растения
  health: 100,                          // Здоровье (0-100%)
  moisture: 50,                         // Ылғалдылық (0-100%)
  temperature: 20,                      // Температура (°C)
  lastWatered: "2025-04-15T14:00:00Z",  // Последний полив
  createdAt: "2025-04-15T14:00:00Z"     // Время создания
}
```

## Сохранение данных

Все данные автоматически сохраняются в **localStorage** браузера под ключом `jasylbaq_plants`.

**Проверить сохраненные данные:**
```javascript
console.log(localStorage.getItem('jasylbaq_plants'));
```

**Очистить все данные:**
```javascript
localStorage.removeItem('jasylbaq_plants');
```

## События и обновлении

После добавления, удаления или обновления растения:

## Примеры использования

### Добавить 3 растения
```javascript
const plants = [
  { name: 'Алма ағашы', type: 'Жеміс ағашы', sector: 'А-01', 
    plantedDate: '2024-03-15', waterFrequency: 'Аптасына бір рет', notes: '' },
  { name: 'Роза', type: 'Гүл', sector: 'Б-02',
    plantedDate: '2024-05-10', waterFrequency: 'Күнде бір рет', notes: 'Красная' },
  { name: 'Кактус', type: 'Шөлді өсімдік', sector: 'В-03',
    plantedDate: '2024-07-20', waterFrequency: '2 күнде бір рет', notes: 'Мало воды' }
];

plants.forEach(p => addPlant(p));
```

### Получить здоровые растения
```javascript
const healthyPlants = getPlantsFromStorage().filter(p => p.health >= 70);
console.log(healthyPlants);
```

### Экспортировать данные
```javascript
const plants = getPlantsFromStorage();
const json = JSON.stringify(plants, null, 2);
console.log(json);
```
